## Apa Itu Discord Connections?
Discord punya fitur keren yang disebut **Connections**, yang memungkinkan kamu menghubungkan berbagai layanan dan akun ke Discord, seperti domain website, YouTube, PlayStation, GitHub, Twitch, Spotify, dan masih banyak lagi.

## Apa Itu .well-known?
`.well-known` adalah direktori standar yang digunakan untuk menyediakan metadata di server web. Dalam konteks ini, Discord menggunakannya untuk memverifikasi bahwa domain yang dimasukkan benar-benar milik pengguna tersebut.

---

### Cara Menautkan Website ke Connections di Discord

- Buka Menu **`Connections`** Discord dengan cara masuk ke akun Discord kamu.
- Klik ikon *`gear`* di sebelah profil kamu (pojok kiri bawah pada Desktop, pojok kanan atas pada Mobile) untuk membuka **`User Settings`**.
- Di menu sebelah kiri, cari dan klik **`Connections`**.

![Preview](/youtube_integration_viewer.jpg)

- Kamu akan melihat daftar layanan yang bisa dihubungkan.

![Preview](/Connections.png)

---

#### Tambahkan Layanan atau URL Website
- Klik ikon layanan yang ingin kamu tautkan. Kalau kamu ingin menautkan domain websitemu, masukkan URL website, setelahnya akan muncul sebuah *`prompt`* verifikasi seperti berikut:
  
![Preview](/Connections-Check.png)

- Salin token verifikasi yang ada pada form `CONTENT`.

---

#### Buat Folder .well-known
- Buat folder baru bernama `.well-known` di dalam folder `public` (Atau `root` proyek).
- Di dalam folder `.well-known`, buat file teks baru bernama `discord` tanpa ekstensi apapun, dan masukkan token verifikasi yang tadi sudah kamu salin ke dalam file tersebut, *`deploy`* website kamu, lalu tekan tombol **`Verify`**.

![Preview](/Connections-Dir.png)

> Setelah ditautkan, akun atau layanan yang kamu tambahkan akan muncul di profil Discordmu. Teman atau anggota komunitas bisa melihatnya jika kamu memilih untuk membagikannya.
