# Integrasi YouTube untuk Konten Kreator

Fitur ini hanya tersedia bagi Mitra YouTube. Jika channel YouTube belum mengaktifkan fitur membership, integrasi channel YouTube ke server Discord tidak dapat dilakukan.

Kamu dapat membuat server Discord khusus untuk membership YouTube atau membuat beberapa channel (atau lebih) yang hanya dapat diakses oleh anggota membership.

Untuk memulai, hubungkan channel YouTube ke akun Discord melalui **`Connections`**. Caranya, ikuti langkah-langkah di **<a href="/posts/menautkan-domain-ke-discord">Menautkan Domain Website ke Discord</a>**, namun pilih ikon `YouTube`.

![Preview](/youtube_integration_viewer.jpg)

Jika YouTube sudah terhubung ke Discord, integrasikan YouTube ke server.

1. Buka server dan klik nama server di pojok kiri atas untuk membuka menu **`Server Settings`**, lalu masuk ke **`Integrations`**. Klik kanan pada nama server juga dapat dilakukan untuk membuka menu yang sama.
2. Ikuti petunjuk yang ada di halaman tersebut untuk mengintegrasikan channel YouTube ke server Discord.

![Preview](/Discord-YT.jpg)

---

# Mengubah Integrasi YouTube

- Setelah diintegrasi, akan muncul role baru yang dibuat secara otomatis khusus untuk anggota membership channelmu, role ini bernama **`YouTube Member`**. Akan ada role tambahan untuk setiap tingkatan membership yang dibuat. Misalnya, jika terdapat tingkatan `"Plus"`, role yang otomatis dibuat bernama **`YouTube Member: Plus`**.

- Nama role ini dapat diganti langsung di Discord. Izinnya juga dapat diatur di **`Server Settings > Roles`**. Warna role pun dapat diubah. 

- Karena role ini diatur secara otomatis oleh integrasi YouTube, kamu tidak dapat menetapkan atau menghapus membership secara manual.

> Jika server kamu memiliki `Rule Screening` untuk anggota baru, mereka dapat melewati aturan tersebut dan langsung bergabung dengan server jika menjadi anggota melalui YouTube Membership.

---

> Jika kamu mengalami masalah dalam mengatur Integrasi YouTube Membership, dukungan Discord dapat dihubungi untuk mendapatkan bantuan: [Integrasi YouTube Membership: FAQ](https://support.discord.com/hc/en-us/articles/215162978-YouTube-Channel-Memberships-Integration-FAQ) (dalam bahasa Inggris).
